# 🔐 PII Detection & Anonymization System

A Flask-based web application that detects and anonymizes Personally Identifiable Information (PII) from text and structured data.

## ✨ Features
- User login and registration
- Admin dashboard to manage users
- PII detection from text and form data
- Anonymization of sensitive information
- Role-based access (Admin / User)
- Vanta.js animated background UI

## 🛠️ Tech Stack
| Layer | Technology |
|-------|-----------|
| Backend | Python, Flask |
| Frontend | HTML, CSS |
| UI Effect | Vanta.js |

## ▶️ How to Run Locally
1. Install dependencies:
```bash
   pip install flask
```
2. Run the app:
```bash
   python backend/app.py
```
3. Open your browser and go to:
  http://127.0.0.1:5000
